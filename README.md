# echobox
ExpressJS container that repeats what you say.

## Versions
- martyca/alpinenodejs 1.0
- ExpressJS 4.17.1
- NodeJS 12.15.0
- NPM 6.13.4

## Instructions
### Run
`docker run -d -p 8080:8080 martyca/echobox`
### Test
`curl -s -X PUT -d "foo=bar" \-H "Accept: application/json" "localhost:8080/foo/bar?foo=bar&baz=qux" | jq .`
